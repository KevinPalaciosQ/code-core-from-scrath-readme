# String: substr()
## Solution
```
function firstWord(str) {
    let fB = str.indexOf(' ');
    return str.substring(0, fB);
  }
```

# String: replace()
## Solution
```
function normalize(DateToday) {
    return DateToday.replace(/-/g, "/");
  }
```

# Farenheit
## Solution
```
function toFahrenheit(getcelsius) {
```

# Boolean
## Solution
```
function nand(x, y) {
    return !(x && y);
  }
```
